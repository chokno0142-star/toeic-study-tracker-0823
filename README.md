# TOEIC Study Tracker 0823

A mini web app to track TOEIC study progress toward the 2026-08-23 exam.

## Purpose

This project is a small web application for managing TOEIC study progress before the TOEIC Listening & Reading test on the morning of 2026-08-23.

The first goal is not to build a large system, but to create a simple browser-based app that can save daily study records.

## Features

- Show the number of days remaining until the TOEIC exam
- Record today's 金フレ study range
- Record today's study time
- Record mock test scores
- Write weakness notes
- Save records using localStorage
- Display saved study records

## Tech Stack

- HTML
- CSS
- JavaScript
- localStorage

## Development Policy

This first version does not include:

- Login
- Database
- AI features
- Graphs
- Backend server

The first milestone is complete when the app works in the browser and can save study records locally.
